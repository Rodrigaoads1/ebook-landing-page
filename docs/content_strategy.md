# Estratégia de Conteúdo e Copywriting: Sistema MAE™

## Pilares de Conteúdo (Use sempre estes 4)

### 1. Educação Estratégica
**Objetivo:** Ensinar conceitos sem jargão  
**Tom:** Professor calmo, não "guru"

**Exemplos de temas:**
- "O que é um funil de vendas (explicado para quem tem alergia a marketing)"
- "Por que profissionais técnicos perdem clientes sem saber"
- "A diferença entre tráfego e tração"
- "Como funciona a jornada do cliente (sem complicação)"

### 2. Desmistificação
**Objetivo:** Quebrar mitos do mercado  
**Tom:** Crítico inteligente, não agressivo

**Exemplos de temas:**
- "Você não precisa de 10K seguidores para vender"
- "Por que 'postar todo dia' não funciona (sem estratégia)"
- "3 coisas que agências dizem e você deveria ignorar"
- "Marketing não é entretenimento"

### 3. Casos e Lógica
**Objetivo:** Mostrar prova social sem exagero  
**Tom:** Analítico, baseado em dados

**Exemplos de temas:**
- "Como um advogado estruturou funil sem virar influencer"
- "De 2 para 12 clientes/mês: a história de um psicólogo"
- "O que mudou quando este arquiteto parou de 'postar aleatoriamente'"
- "Estudo de caso: consultoria financeira em 60 dias"

### 4. Antipadrões
**Objetivo:** Mostrar o que NÃO fazer  
**Tom:** Sincero, mas respeitoso

**Exemplos de temas:**
- "3 erros que mantêm você dependendo de indicação"
- "Por que você está fazendo marketing errado (e não é culpa sua)"
- "O que eu faria se tivesse que recomeçar do zero"
- "Sinais de que sua estratégia não está funcionando"

## Tom de Voz e Linguagem

### O Que É (Sempre)

✅ **Empático:** "Eu sei que você já tentou e não funcionou"  
✅ **Calmo:** "Vamos por partes, sem correria"  
✅ **Inteligente:** "Vou te explicar a lógica por trás disso"  
✅ **Direto:** "Isso funciona. Aquilo não funciona. Simples assim."  
✅ **Respeitoso:** "Você é excelente tecnicamente, só não aprendeu isso ainda"

### O Que NÃO É (Nunca)

❌ **Motivacional barato:** "Você consegue!", "Bora!", "Vai!"  
❌ **Agressivo:** "Se você não fizer isso, vai falir"  
❌ **Hype:** "MÉTODO SECRETO!", "FÓRMULA MÁGICA!"  
❌ **Complexo:** Jargões desnecessários, termos técnicos sem explicação  
❌ **Empurrador:** "Última chance!", "Só hoje!"

### Frases-Chave (Use com frequência)

- "A faculdade não te ensinou isso — e não é culpa sua"
- "Vender não é se expor, é estruturar"
- "Sem funil, não existe previsibilidade"
- "Você não precisa virar influencer para ter clientes"
- "Marketing sem estratégia é só barulho"
- "Sistema, não sorte"
- "Autoridade silenciosa"

## Calendário de Conteúdo (Modelo Semanal)

### Segunda-feira: Educação Estratégica
**Formato:** Carrossel de 5-8 slides  
**Exemplo:** "Como funciona um funil (sem complicação)"

### Terça-feira: Desmistificação
**Formato:** Post de texto + imagem simples  
**Exemplo:** "Você não precisa de 10K seguidores para vender — precisa de SISTEMA"

### Quarta-feira: Caso/Lógica
**Formato:** Carrossel ou vídeo curto  
**Exemplo:** "Como um dentista foi de 5 para 18 pacientes/mês"

### Quinta-feira: Antipadrão
**Formato:** Post de texto  
**Exemplo:** "3 erros que mantêm você dependendo de indicação"

### Sexta-feira: Educação (Recap ou Aprofundamento)
**Formato:** Vídeo ou carrossel  
**Exemplo:** "Os 3 pilares do Sistema MAE™ (resumo visual)"

### Sábado (Opcional): Bastidores ou Mentalidade
**Formato:** Story ou post leve  
**Exemplo:** "O que eu aprendi esta semana trabalhando com profissionais liberais"

### Domingo: Descanso ou Repost
Reutilize o melhor conteúdo da semana


## Estrutura de CTA (Call-to-Action)

### Hierarquia de CTAs

**CTA PRIMÁRIO (mais comum):**
```
"Baixe o checklist gratuito"
"Assista a aula secreta"
"Agende seu diagnóstico"
```

**CTA SECUNDÁRIO (para quem já conhece você):**
```
"Comenta 'CHECKLIST' que eu te mando"
"Me manda um DM com a palavra 'CLAREZA'"
"Responde este e-mail se quiser conversar"
```

**CTA TERCIÁRIO (mais direto):**
```
"Vamos mapear sua situação? Link na bio"
"Pronto para estruturar? Chama no WhatsApp"
```

### Quando Usar Cada CTA

- **Início do funil:** CTA suave (isca gratuita)
- **Meio do funil:** CTA educacional (aula, checklist)
- **Fundo do funil:** CTA direto (diagnóstico, conversa)

## Conteúdo Longo (Blog/LinkedIn Articles)

### Template de Artigo de Autoridade

**ESTRUTURA:**

1. **HOOK (Gancho Inicial)**
```
"Se você é [profissão] e depende de indicação,
este artigo pode mudar como você atrai clientes."
```

2. **PROBLEMA (Identificação)**
```
A maioria dos profissionais liberais enfrenta o mesmo dilema:
→ São excelentes tecnicamente
→ Trabalham muito
→ Mas não têm controle sobre quantos clientes entram

Não é culpa sua.
```

3. **AGITAÇÃO (Amplificar a dor, sem exagero)**
```
Isso gera:
- Ansiedade (não saber se o mês vai ser bom)
- Frustração (já tentou "postar" e não funcionou)
- Vergonha (sente que deveria saber vender)
```

4. **SOLUÇÃO (Sistema, não produto)**
```
A solução não é "postar mais".
É ter um SISTEMA de aquisição previsível.

Isso se chama funil estruturado.
```

5. **EXPLICAÇÃO (Educar)**
```
Um funil é simplesmente:

1. Pessoa te descobre
2. Recebe valor (isca, aula, diagnóstico)
3. Entra em conversa
4. Fecha (se fizer sentido)

Simples. Mas poucos fazem.
```

6. **PROVA (Caso ou lógica)**
```
Exemplo: psicólogo que fiz diagnóstico semana passada.
8 anos de experiência, mas 2-3 clientes/mês.

Estruturamos funil básico.
60 dias depois: 12 clientes/mês.

Diferença: sistema vs. sorte.
```

7. **CTA (Convite não invasivo)**
```
Se você quer mapear sua situação,
agende um Diagnóstico Estratégico.

90 minutos de clareza total.
Link: [LINK]
```

## Resumo: Checklist de Conteúdo

Antes de publicar qualquer conteúdo, pergunte:

✅ Está educando sem complicar?  
✅ Está tirando culpa/vergonha?  
✅ Está posicionando autoridade (não hype)?  
✅ Tem CTA claro e não invasivo?  
✅ Respeita o perfil do profissional liberal?  
✅ Reforça o Sistema MAE™?

Se sim para tudo: PUBLIQUE.  
Se não: AJUSTE.
